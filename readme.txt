EgoHands is a python library to understand the movement of the hands when recorded by a wearable camera (usually head or chest mounted). This video perspective is commonly known as First Person Video (FPV) or just as egovision. Please feel free to use EgoHands for academic purposes but remember to cite our work!

Install
-------
0. install virtualenvwrapper

1. mkvirtual open_egohads

2. pip install -r requirements.py

3. sudo apt-get install python-opencv

4. ln -s /usr/lib/python2.7/dist-packages/cv* ~/.virtualenvs/open_egohands/lib/python2.7/site-packages/.
